Descripción

Este es un bundle especialmente diseñado para que puedas entrar en un ambiente totalmente práctico y de desarrollo en el que a través del uso de la tecnología plugdo™ se hará más sencillo desarrollar cualquier proyecto que necesite NODE como lenguaje de programación. 

Al crear tu bundle podrás usar la plataforma en un ambiente de desarrollo real. Sigue estos pasos que tu instalación sea exitosa, te recomendamos: 

1. Crea una cuenta gratuita en el Cloud Computing From Amazon, si no tienes una cuenta, haz clic aquí para entrar. 

2. Crea una cuenta en el repositorio de GItHub para que guardes tu proyecto y lo puedas presentar en tu portafolio. 

Al tener estos datos, lo pediremos sólo para que tu uso de la plataforma y tu desarrollo, es la misma plataforma la que genera esta integración. No guardamos contraseñas, así que puedes estar tranquilo. Lee nuestros Términos y Condiciones. 

¿Que incluye el bundle de plugdo NODE?

  • 1 Workspace para que trabajes en tu proyecto y desarrolles en la plataforma. 
  • 1 Base de datos integrada MYSQL 5.7.
    Nota: Para conectar con la base de datos predeterminada usa estos datos: usuario: root y la contraseña: plugdo
  • 1 Workspace Adminer, que es un administrador de base de datos de MYSQL.
    Nota: Para acceder a este workspace usa el puerto 81.
